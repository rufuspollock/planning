---
title: Google Analytics to CSV
description: Automated export of Google Analytics data to CSV files in a GitHub repository via Actions.
created: 2026-01-28
---

A system that uses GitHub Actions to regularly retrieve Google Analytics data and store it as CSV files in a repository. The goal is data liberation - getting analytics out of Google's interface into a portable, version-controlled format. Dashboards are optional; the core value is owning the raw data.
